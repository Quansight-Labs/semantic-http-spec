Semantic HTTP ResourceÂ¶

A conforming Semantic HTTP Resource is a URL-addressable resource that satisfies the following requirements:

  • MUST provide a Schema Link Header for any HTTP response that returns an entity body.

  • MUST provide a RDF Context Link Header for any HTTP response that returns an entity body.

  • MUST support the HTTP OPTIONS method and provide a Semantic HTTP OPTIONS entity as the response body for an OPTIONS request. HTTP OPTIONS responses MUST also include the Schema Link Header and RDF Context Link Header.
